## 부서별 발언 요약

### 발행인
- 웹 론칭을 "이벤트형 론칭"으로 — 무료 7일 시리즈를 "첫 번째 구독자 증정" 캠페인으로 포장
- 초기 파트너십 타겟: 링커리어, 스펙업, 취업 커뮤니티, 자기계발 유튜버, 대학 취창업센터
- 스폰서 광고 슬롯은 초기 100명 사용자 데이터 확보 후 세일즈 시작

### 편집장
- 론칭 전 샘플 신문 5종 필수 (의사/개발자/크리에이터/창업가/해외취업 꿈 주인공)
- 신문 품질 기준서 v1.0 이번 주 확정 — 1인칭 현재진행형, 미래형 표현 금지 원칙 문서화
- 기본 의뢰(한 줄)로도 완성도 높은 1호 신문 생성 가능 여부 사전 품질 검수

### 기자
- 샘플 신문 = PDF 다운로드 + SNS 카드뉴스 2가지 포맷으로 제작
- 신문 1면 헤드라인 → SNS 공유 자동 썸네일 이미지 생성 기능 제안
- 가상 기자 바이라인(byline) 도입으로 서비스 인격 부여

### 마케팅 디렉터
- 채널 전략: 광고비 집행 X → 유기적 채널(인스타/스레드/틱톡/네이버 블로그) 집중
- 핵심 훅: "10년 후 나는 무엇을 하고 있을까?" — 불안 → 설렘으로 전환
- 론칭 첫 주 100명 무료 체험단 모집 캠페인

### 콘텐츠 디렉터
- SNS 콘텐츠 3유형: 감성 카드 / 교육 콘텐츠 / 체험단 후기
- 네이버 블로그: SEO 채널로 롱테일 키워드 포스팅 주 2회
- 틱톡/릴스: 실사용자 신문 낭독 숏폼 → 바이럴 핵심

### 광고영업
- "파운딩 스폰서" 프로그램: 초기 10개 기업 한정 1호 타이틀 + 할인, 실 노출은 500명 달성 후
- 타겟 업종 1순위: 자격증 교육기관(패스원, 에듀윌), 어학원, 취업 컨설팅
- 제안서에 꿈 카테고리별 사용자 분포 예시 데이터 필수 포함

### UI 디자이너
- 신문 UI: 종이 신문 질감 + 모바일 스크롤 최적화 동시 달성
- 공유 이미지 자동 생성 (9:16 비율, 인스타 스토리용) 개발 우선순위 상향
- 온보딩 플로우 5단계 이내로 압축

### 웹 디자이너
- 랜딩페이지 구조: 히어로 → 샘플 신문 → 작동 방식 3단계 → 후기 → CTA
- 히어로 훅 문장 확정: **"10년 후 당신이 살고 있을 하루, 지금 받아보세요."**
- Lighthouse 90점 이상 + SEO 메타태그/OG 이미지/구조화 데이터 완비

### 브랜드 마케터
- 포지셔닝: "자기계발 앱" X → "나만을 위한 미래 콘텐츠 구독"
- 톤앤매너: 진지하지만 따뜻함, 격식체 + 독자가 주인공이 되는 온기
- 모든 접점(로고/앱 아이콘/이메일 발신자)에서 "꿈신문사 편집국"으로 통일

### HR매니저
- 전 직원이 마케터 — 직원 공유 가이드라인 제작 (강요 X, 자랑스럽게 공유 가능한 포맷)
- 체험단 온보딩 FAQ 문서 + 막히는 지점 사전 매핑
- 첫 신문 수령 24시간 내 피드백 요청 → 7일 완료 후 후기 작성 요청 플로우
