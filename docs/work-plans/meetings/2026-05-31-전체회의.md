## 부서별 발언 요약

### 기자
- 개발자 편 완성, 크리에이터 편 기사 2~3 오늘 완성 목표
- 하루 1~2편 속도 → 6/15까지 스탁 원고 최소 5편 필요
- 유튜버 꿈 원고도 착수 (체험단 레퍼런스용)

### 편집장
- 품질 기준서 3원칙 확정: ① 미래형 절대 금지 ② 날짜 2030년 이후 ③ 1인칭 이미 살고 있는 일상
- 기자단 자가 체크리스트 배포로 검수 병목 분산
- 오늘 오후 3시 샘플 신문 1호 초안 공유 목표 설정

### 브랜드 마케터
- 브랜드 아이덴티티 3계층: 핵심(나의 미래를 읽는다) / 기능(기자단이 씁니다) / 감성(설레는 월요일 아침)
- 폰트: Noto Serif KR(헤드라인) + Pretendard(본문) 절충안 제안 → 전원 채택
- 금지 단어 전 채널 적용 점검 오늘 진행

### 웹 디자이너
- CSS 변수로 폰트 두 가지 준비 완료 → 결정 즉시 5분 내 반영 가능
- 샘플 미리보기 + FAQ 섹션 오늘 퍼블리싱 목표
- 도메인 확정 시 canonical URL 메타태그 같이 반영 요청

### UI 디자이너
- 신문 3단 컬럼 모바일 깨짐 현상 발견 → 오후 CTO와 수정
- 인스타 공유 템플릿 추가 2종 제작 (스토리 1080×1920, 인용구 카드)
- 폰트 확정 후 브랜드 가이드 즉시 업데이트

### 마케팅 디렉터
- 나노 인플루언서 기준: 팔로워 1K~10K, 꿈/커리어/자기계발 카테고리
