## 합의 사항

### 1. 소셜 바이오 문구 — C안 확정
> **"2035년 당신의 일상이 궁금하세요? 꿈신문사 기자단이 씁니다"**

이유: "AI" 없음, "기자단" 브랜딩 강화, 구체적 연도로 서비스 콘셉트 즉각 전달

---

### 2. 체험단 모집 플랫폼 — 확정
- **인스타 스토리 신청폼(구글폼)** — 즉시 실행, 비용 0
- **링커리어 병행** — 취준/직장인 정확 타겟
- 에브리타임: 유료 확인 시 이번 라운드 제외

---

### 3. 신문 공유 이미지 자동 생성 — MVP 제외, Phase 2 이관
- 대신: UI 디자이너 정적 템플릿 제공 → 사용자 스크린샷 공유 임시 운영
- OG 이미지(랜딩페이지 정적)는 웹 디자이너가 오늘 완성

---

### 4. "AI" 표현 공식 정책 확정
> FAQ 답변: **"꿈신문사는 편집국 시스템과 기자단이 함께 작성합니다. 외부에 AI 여부를 별도 표기하지 않습니다."**

- 전체 공개 문서에서 "AI", "인공지능", "챗봇" 단어 삭제
- FAQ에 질문은 유지(독자가 궁금해하므로), 답변만 위 문구로 통일
