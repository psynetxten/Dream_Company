## 확정 구현 스펙

### 화면 흐름
```
[접속]
  → 신문 제호 "꿈신문사" 0.5초 페이드인
  → 타이핑 시작 (40~60ms/글자)
     "2035년 봄, _____씨의 이름이 처음으로 뉴스 첫 면에 실렸다."
  → 빈칸에 커서 자동 포커스 + 0.5초 주기 깜빡임
  → 사용자 이름 타이핑 (문장 안에서 실시간 완성)
  → Enter → 다음 줄 타이핑 + 백그라운드 미리보기 생성 시작
  → "이 기사 전체를 받아보시겠어요?"
  → 이메일 입력 (Magic Link 방식, 비밀번호 없음)
  → 발송 완료 + SNS 공유 카드 자동 생성
```

### 확정 타이핑 문장
> **"2035년 봄, _____씨의 이름이 처음으로 뉴스 첫 면에 실렸다."**

### 디자인 스펙
- 배경: 신문지 질감 (아이보리 #F7F4EE / 세피아 톤)
- 폰트: Noto Serif KR (신문체)
- 타이핑 속도: 40~60ms/글자
- 커서: 세로 막대(|) 0.5초 깜빡임
- 로고: 최상단 마스트헤드(신문 제호) 형식

### 소프트 가입 플로우 (수정 사항)
- 이름만 입력 → 즉시 미리보기 (가입 아님)
- "전체 받기" → 이메일 입력 → Magic Link 발송 → 가입 확정
- 법적 동의: 이메일 입력 화면에서 체크박스 1개로 처리
